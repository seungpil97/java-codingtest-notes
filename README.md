# Algorithm Study

백엔드 취업 준비를 위해 코딩테스트 문제 풀이와 면접 설명 연습을 함께 기록하는 저장소입니다.

단순히 정답 코드만 작성하는 것이 아니라,  
다음 내용을 함께 정리하는 것을 목표로 합니다.

- 문제 풀이 코드
- 풀이 아이디어
- 시간복잡도 / 공간복잡도
- 관련 면접 질문과 답변
- 짧은 회고

## 목표
- 코딩테스트 기초 체력 만들기
- 문제 풀이를 말로 설명하는 연습
- 백엔드 면접에서 나올 수 있는 기본 개념 연결

## 진행 방식
- 하루 1~3문제 풀이
- 문제마다 Java 코드 작성
- 문제별 핵심 면접 질문 정리
- 주차별 README로 학습 기록 정리
